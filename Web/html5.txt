HTML基础讲解：
1. 声明：<!DOCTYPE>
	HTML有许多不同版本，该声明告诉浏览器确切的HTML版本，以防显示页面出错。
2. HTML基础标签
    html  :  包括head部分和body部分，可以声明显示文字
    head ：头部，可以声明编码格式，以及标题
    body ：页面显示内容
3. HTML标题
    <h1>...<h6>等标签进行定义
4. HTML段落
    <p>来表示段落
5. HTML链接
     <a>标签定义链接
 	<a href="http://www.jikexueyuan.com">学习加油</a>
6.HTML图像
     <img>标签定义图像
	<img src="picture/1.jpg">
7.HTML元素
    元素指的是从开始标签到结束标签的所有代码<p> 。。。</p>
    <br/>这是一个空标签（空元素）（换行）  
8.HTML元素语法：
    元素内容是开始标签与结束标签之间的内容
    空元素再开始标签中进行关闭
    大多HTML元素可拥有属性
9.嵌套的HTML元素
    大多数元素是可以嵌套的
10.HTML属性
    标签可以拥有属性为元素提供更多的信息
    属性以键/值对的形式出现
    	如： href="www.jikexueyuan.com"
    常用标签属性:
    	<h1>：align对齐方式
	<body>:bgcolor背景颜色
	<a>:target规定在何处打开链接
    通用属性：
	class:规定元素的类名
	id    :规定元素的唯一id
	style:规定元素的样式
	title:规定元素的额外信息
    属性一般在标签内部
11.HTML格式化
    许多定义字体的标签

12.HTML样式
    三种样式的插入方法：
	外部样式表:
		<link rel= "stylesheet" type ="text/css" href = "xxx.css">
	内部样式表:
		<style type = "text/css">
		body {background-color:red}
		p{margin-left:20px}
		</style>
	内联样式表:
		<p style = "color:red">
	这三种插入方法一般都放在<head>标签内部

13. HTML链接
	<a href="xxx">点击我</a> 用文本作为链接
	<a href="http://www.jikexueyuan.com"><img src="picture/1.jpg"></a>用图片进行链接
	    
	<a name="tip">hello</a>
    	<a href="#tip">dianwo</a> 文件内部跳转

14. HTML表格 
	<table></table> 添加表格

15. HTML列表
	<ul></ul>	无序列表
	<ol></ol>	有序列表
	<dl></dl>	自定义列表

16. HTML块
	块元素：（显示时通常以新行开始）
		<p></p>
		<h1></h1>
	内联元素: （不会以新行开始）
		<b> <a> <img>
	<div>元素:组合HTML元素的容器
	<sapn>元素:只能文本
17. HTML布局
	一般用<div>元素进行布局
	<table>元素进行布局

18. HTML表单:（重中之重）
	用于获取不同类型的用户输入
	（数字，字符等等）
	表单元素的新增属性:
19. HTML php坏境搭建
	php与html5交互（前后端交互）
		GET：密码会显示在网址栏上，不安全，但是可以资源定位
		POST：密码等内容不会显示在网址栏上，安全，但是不能资源定位
20. HTML 框架
	iframe内联框架（实现页面间的嵌套）
21. HTML 实体
	实体：HTML中预留字符必须被保存为字符实体如 < > & 都不能显现
	要想现实特殊字符， 需要用 &gt 的形式
	
22. HTML全局属性
	所有元素都可以使用的属性

23. HTML新增主体元素
	article元素
	section元素
	nav元素（导航
	aside元素
